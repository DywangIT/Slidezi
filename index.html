Slidezi.js

A lightweight and simple slide library built with pure JavaScript.

Try It Out

Slidezi is a lightweight slide library built with pure JavaScript. (Source code available on GitHub)

  • No dependencies
  • Fully customizable with CSS
  • Supports URL state persistence
  • Simple and intuitive API
  • Lightweight and optimized

Here are some examples of how to use Slidezi:

Installation

Method Steps Direct Download Download ZIP

Simply extract the ZIP file and use the slidezi.min.js and slidezi.min.css files in your project.

Slidezi provides a small CSS file that you can include directly via a CDN link. This makes it easy to integrate into your project and customize as needed.

Include the Slidezi JavaScript file in your project:

<script src="https://cdn.jsdelivr.net/gh/DywangIT/slidezi@v1.0.0/slidezi.min.js"></script>
You can use the released versions of Slidezi available here.

Usage

Here’s the basic HTML structure you’ll need to set up Slidezi: 

<div id="my-slider">
    <div>Slide 1</div>
    <div>Slide 2</div>
    <div>Slide 3</div>
    <div>Slide 4</div>
    <div>Slide 5</div>
    <div>Slide 6</div>
    <div>Slide 7</div>
</div>

And here’s how you can use it with JavaScript:

const mySlider = new Slidezi("#my-slider", {
    items: 2, // The number of items shown on the screen
    loop: true, // Enable loop after last item (1 -> 2 -> 3 -> 1)
    speed: 500, // Transition speed
    nav: true // Show some dots and active dots below the slide
});

Options

Option Type Description
items number The number of items shown on the screen (default: 3)
loop boolean Enable loop after last item (default: false)
speed number Slide transition speed(default: 300ms)
nav boolean Show some dots and active dots below the slide (default: true)
