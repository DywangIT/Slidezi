Installation

Method Steps Direct Download Download ZIP

Simply extract the ZIP file and use the slidezi.min.js and slidezi.min.css files in your project.

Slidezi provides a small CSS file that you can include directly via a CDN link. This makes it easy to integrate into your project and customize as needed.

Include the Slidezi JavaScript file in your project:

<script src="https://cdn.jsdelivr.net/gh/DywangIT/slidezi@v1.0.0/slidezi.min.js"></script>
You can use the released versions of Slidezi available here.

Usage

Here’s the basic HTML structure you’ll need to set up Slidezi:

<ul id="my-slides">
    <li><a href="#panel1">Slide 1</a></li>
    <li><a href="#panel2">Slide 2</a></li>
</ul>
<div id="panel1">Content 1</div>
<div id="panel2">Content 2</div>

And here’s how you can use it with JavaScript:

// Initialize a new slide instance
const slides = new Slidezi('#my-slides', {
    activeClassName: 'slidezi--active',
    remember: true, // Keeps the active slide in the URL
    onChange: function({ slide, panel }) {
        console.log(`Switched to ${slide.textContent}`);
    }
});

// Switch to a specific slide
slides.switch('#panel2');

// Destroy the slide instance
slides.destroy();

Options

Option Type Description activeClassName string CSS class applied to the active slide (default: 'slidezi--active') remember boolean Persists the active slide state in the URL (default: false) onChange function A callback function that runs whenever the active slide changes. It receives an object with two properties: slide (the DOM element of the newly active slide) and panel (the DOM element of the corresponding content panel). By default, it’s set to null, meaning no action is taken unless you define a function.

API

Method Description switch(input) Switches to a specific slide based on the provided input. The input can be either: (1) a string representing the href attribute of a slide (e.g., '#panel2'), or (2) a DOM element that is one of the slide links (e.g., an <a> element from the slides list). destroy() Removes all Slidezi functionality, restores the original HTML, and unbinds events.
